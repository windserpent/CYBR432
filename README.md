Python 3.11 or greater is required.

To start the application:
-Start a terminal window or Powershell window
-Browse to the directory where the application files are stored
-Depending on your Python installation, run command:
py main.py
-or:
python main.py

To send a message:
-Configure the recipient's IP address
-Start the listener
-Select "Send Message"

To check for received messages:
-Select "View Mailbox"
